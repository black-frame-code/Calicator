# 🧮 Calculator A.1

A modern calculator built with pure **HTML, CSS, and JavaScript**.

## ✨ Features

- ➕ Basic calculations
  - Addition
  - Subtraction
  - Multiplication
  - Division

- 🧠 Advanced functions
  - Square Root (√)
  - Power (^)
  - Percentage (%)
  - sin()
  - cos()
  - tan()
  - log()
  - ln()
  - π (Pi)
  - e (Euler's Number)

- ⌫ Backspace button

- 🌓 Dark / Light Theme

- 📜 Calculation History
  - Automatically saves history
  - History remains after page refresh
  - Can be cleared manually

- 📱 Responsive Design
  - Works on Desktop
  - Works on Mobile

---

## 🚀 Getting Started

1. Download the project.
2. Open `index.html` in your browser.

No installation required.
